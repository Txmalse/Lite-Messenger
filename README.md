# 💬 Python Socket Chat (CustomTkinter)

Простий графічний чат-клієнт на Python з використанням **socket**, **threading** та **CustomTkinter**.

Додаток дозволяє підключатися до TCP сервера, надсилати повідомлення та отримувати їх у реальному часі через графічний інтерфейс.

---

# 🚀 Функції

- підключення до TCP чат-сервера
- відправка текстових повідомлень
- отримання повідомлень у реальному часі
- окремий потік для отримання повідомлень
- графічний інтерфейс на **CustomTkinter**
- адаптивний UI
- анімоване бокове меню
- підтримка різних типів повідомлень:
  - TEXT
  - IMAGE

# ⚙️ Як працює чат

Клієнт використовує **TCP сокет** для звʼязку з сервером.

### Формат повідомлень

```
TYPE@AUTHOR@MESSAGE
```

Приклад:

```
TEXT@Maksym@Hello everyone
```

### Підтримувані типи

| Тип | Опис |
|----|----|
| TEXT | текстове повідомлення |
| IMAGE | повідомлення про зображення |

# 🧠 Архітектура

Основні компоненти:

### `MainWindow`

Головне вікно програми, яке:

- створює UI
- підключається до сервера
- керує повідомленнями
- обробляє події інтерфейсу

---

### Потік отримання повідомлень

Для отримання повідомлень використовується окремий потік:

```python
threading.Thread(target=self.recv_message, daemon=True).start()
```

Це дозволяє:

- не блокувати UI
- отримувати повідомлення у фоновому режимі

---

### Буфер повідомлень

Сервер може відправляти повідомлення частинами, тому використовується **buffer**.

```
buffer += chunk.decode()
```

Коли знаходиться `\n`, повідомлення обробляється.
